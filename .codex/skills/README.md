# Repo-Local Skills

This directory contains the skills that should travel with the Hudson Hustle repo.

## Vendored Generic Skills

These are repo-local copies of general design-review skills. They are intentionally kept here so the project can carry its preferred design tooling with the repo instead of depending on one developer's home-level Codex setup.

- `audit`
- `critique`
- `distill`
- `impeccable`
- `layout`
- `polish`
- `typeset`

Do not customize these lightly. If Hudson Hustle needs project-specific guidance, prefer documenting that guidance in repo-specific skills or in product/design docs rather than forking the generic skill behavior.

## Maintenance Guidance

- If a vendored generic skill is refreshed from upstream, remove any one-time installer or cleanup instructions after they have been applied.
- If a Hudson Hustle-specific workflow changes, update the project-specific skill here rather than patching the vendored generic skills.

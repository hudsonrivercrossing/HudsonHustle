# Map And Balance Notes, March 2026

## Why This Pass Happened
- The first board read more like a generic node graph than a recognizably NYC/NJ transit game.
- Several west-east routes made cross-river play too loose, especially around World Trade and the Metropark to Jamaica shortcut.
- The visual system for cards and tickets was functional, but it did not yet feel like a cohesive board-game product.

## Data Changes
- Repositioned hubs to better reflect the real regional shape: New Jersey west, Manhattan narrow and vertical, Brooklyn south, Queens east.
- Renamed a few labels toward real transit language, including `Penn District` and `Queens Plaza`.
- Removed redundant shortcuts:
  - `jersey-city -> world-trade`
  - `newport -> chelsea`
  - `newport -> world-trade`
  - `metropark -> jamaica`
  - `greenpoint -> court-square`
- The current board now sits at `31` cities and `62` routes, which is inside the intended `28-32` / `55-65` target.

## Balance Reasons
- Removing extra World Trade and Newport shortcuts restores more pressure to Hudson crossings instead of letting lower Manhattan absorb too many efficient paths.
- Removing `metropark -> jamaica` makes outer New Jersey to Queens/Brooklyn tickets ask for more intentional planning instead of one diagonal express line.
- Keeping both `secaucus -> penn district` tunnels and both `exchange place -> world trade` tunnels preserves high-value choke points without flattening the whole board.
- Ticket values were adjusted only where the old numbers were now clearly too rich or too flat after the route changes.

## Visual System Changes
- The board now uses shoreline cues, region labels, and segmented route pieces so it reads more like a crafted board map.
- Ferry and tunnel routes are marked directly on the board with small badges.
- Train cards, market cards, and destination tickets now share a more tactile paper-card treatment.

## Focused Playtest
- Run one `4-player` game and watch whether Hudson crossings around `Penn District`, `Exchange Place`, and `World Trade` feel contested but not impossible.
- Run one `2-player` game and check whether outer tickets like `Rahway -> Jamaica` and `Atlantic Terminal -> Secaucus` still feel worth keeping.

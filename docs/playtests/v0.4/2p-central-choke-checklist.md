# v0.4 2-Player Central Choke Checklist

Historical note:
- this checklist was written for the `v0.4-next-wave` draft-phase review
- keep that naming as a historical reference point, not as the current release label

## Purpose
- Run one focused `2-player` manual playtest.
- Check whether too many game plans collapse into the same central Manhattan core.
- Record observations before changing data.

## Watch List
- `Penn District`
- `World Trade`
- `Battery Park`
- `Grand Central`
- `Union Square`
- `Long Isl. City`

## Before You Start
- Use the current `v0.4-next-wave` draft.
- Play a normal `2-player` game.
- Do not force weird moves just to hit the checklist.
- Record notes after each phase, not only at the end.

## Setup Snapshot
- Write down each playerŌĆÖs starting tickets.
- Mark whether each playerŌĆÖs opening tickets naturally pull them toward:
  - west side
  - central Manhattan
  - Brooklyn
  - Queens
  - outer nodes

## Opening Turns Checklist
- Did either player immediately prioritize:
  - `Penn District`
  - `World Trade`
  - `Battery Park`
- Did the first 3-4 turns already make the middle look mandatory?
- Did either player feel they had a viable opening plan that avoided the central core?

## Midgame Checklist
- By the middle of the game, record:
  - which player owns `Penn District` routes
  - which player owns `World Trade` routes
  - which player owns `Battery Park` routes
- Ask:
  - are both players still being pulled through the same central ladder?
  - does one playerŌĆÖs central claim make several different tickets harder at once?
  - does `Battery Park` rescue too many otherwise difficult lines?
  - does `Union Square` matter in actual decisions, or is it mostly optional?

## Ticket Pressure Checklist
- For each player, note:
  - which ticket was easiest to complete
  - which ticket felt hardest
  - whether the hard ticket was difficult because of real competition or because of bad draw luck
- Ask:
  - do different tickets still route through the same central trunk?
  - do `Exchange Pl. -> Jamaica`, `World Trade -> Downtown Bklyn`, `World Trade -> Grand Central`, and `Newark -> Grand Central` feel too similar in practical solution?

## Choke Point Checklist
- Mark whether these spaces felt contested:
  - `Penn District`
  - `World Trade`
  - `Battery Park`
  - `Grand Central`
  - `Long Isl. City`
- For each contested node, ask:
  - was the contest interesting?
  - or did it feel mandatory and repetitive?

## Station Checklist
- Did either player build a station?
- If no:
  - was that because stations are weak
  - or because the current graph was easy enough without them
- If yes:
  - did the station feel like a meaningful rescue
  - or a routine obvious choice

## Endgame Checklist
- When the game ended, ask:
  - did the winner mostly win through central access?
  - did longest route matter?
  - did outer nodes matter meaningfully?
  - were any routes or tickets effectively ignored all game?

## Quick Scorecard
- Central choke pressure:
  - `Low / Medium / High`
- `Battery Park` dominance:
  - `Low / Medium / High`
- Ticket overlap around same trunk:
  - `Low / Medium / High`
- Station usefulness:
  - `Weak / Fair / Strong`
- Outer-node relevance:
  - `Weak / Fair / Strong`

## If You See These Signals
- Both players repeatedly forced into `Penn District -> World Trade -> Battery Park`
  - likely central overconcentration
- `Battery Park` solves too many ticket problems
  - likely too flexible
- `Union Square` never matters
  - likely under-supported
- `Flushing`, `Red Hook`, or `Newark Airport` never matter
  - likely under-ticketed or too disconnected from core goals

## After The Test
- Summarize:
  - top 3 strongest choke points
  - top 3 weakest or most decorative stations
  - one route to weaken
  - one route or ticket to strengthen

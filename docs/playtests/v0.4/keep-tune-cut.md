# v0.4 Keep / Tune / Cut

Historical note:
- this triage uses the draft-era name `v0.4-next-wave`
- it documents the tuning phase that led toward the later frozen `v0.4-flushing-newark-airport` release

## Purpose
- Turn the current `v0.4-next-wave` review into a concrete triage list.
- Separate what already feels right from what should be tuned next and what is most expendable.
- Focus on the current draft, not on hypothetical future stations.

## Keep

### Routes To Keep
- `secaucus -> midtown-west`
  - still a strong trans-Hudson identity even as a single tunnel trunk
- `secaucus -> newark-penn`
  - now at length `3`, this is a healthier west-side commitment than the earlier cheap south-west hop
- `exchange-place -> world-trade` double route
  - high flavor, strong real-world grounding, and good local choke value
- `newark-penn -> grove-st -> exchange-place`
  - better west-side layering than the old single jump to Exchange Pl.
- `secaucus -> hoboken`
  - important west-side branch identity that keeps Hoboken from collapsing into a pure support node
- `midtown-west -> grand-central`
  - essential Manhattan trunk
- `grand-central -> flushing`
  - one of the healthiest outer-node routes in the draft
- `flushing -> jamaica`
  - gives eastern Queens a clean outer cap
- `hudson-yards -> grand-central`
  - supports a west-Manhattan layer without overgrowing the board or duplicating Penn District too directly
- `world-trade -> battery-park`
  - south-Manhattan layering is better with this than with a single WTC-to-Brooklyn jump
- `battery-park -> downtown-brooklyn`
  - makes the harbor-side spine legible and strategically relevant
- `battery-park -> red-hook`
  - good board-space usage and clear flavor identity
- `red-hook -> atlantic-terminal`
  - keeps Red Hook from being a dead decorative leaf
- `long-island-city -> williamsburg`
  - strong borough adjacency and useful bridge into Brooklyn play
- `union-square -> williamsburg`
  - gives Union Square a more distinctive Manhattan/Brooklyn role than the old direct World Trade support edge
- `williamsburg -> downtown-brooklyn`
  - one of the cleaner borough-internal connectors
- `grand-central -> long-island-city`
  - useful midtown-to-Queens bridge without the older LIC-to-Downtown shortcut
- `grand-central -> jamaica`
  - still one of the map's signature outer-core trunks and now does more work after recent realism cleanup

### Tickets To Keep
- `t-grand-central-jamaica`
  - iconic Queens trunk goal with clear strategy
- `t-grand-central-flushing`
  - now a better outer-Queens incentive after the recent points increase
- `t-flushing-jamaica`
  - local outer-Queens fight that does not overload Manhattan
- `t-exchange-place-downtown-brooklyn`
  - good cross-river long ticket that no longer leans on the removed Newark -> WTC shortcut
- `t-hudson-yards-flushing`
  - one of the best new outer-node tickets because it rewards the west-Manhattan/east branch, not the Battery Park trunk
- `t-red-hook-jamaica`
  - healthy outer-borough stretch that promotes non-central routing
- `t-flushing-atlantic-terminal`
  - good east-to-south outer ticket that pushes players off the same Manhattan middle
- `t-newark-airport-exchange-place`
  - validates Newark Airport without forcing Battery Park involvement
- `t-secaucus-world-trade`
  - still a good cross-river long ticket without being as over-direct as Newark -> Grand Central
- `t-secaucus-downtown-brooklyn`
  - good cross-region long ticket with multiple viable lines
- `t-newark-downtown-brooklyn`
  - still a strong long objective that does not rely on the removed Newark -> WTC shortcut
- `t-hoboken-grove-st`
  - useful local ticket that validates the Jersey corridor
- `t-red-hook-atlantic-terminal`
  - helps justify the harbor branch
- `newark-airport -> hudson-yards`
  - a strong flavor route with real difficulty, but it should be kept only if the helicopter-style tunnel feels challenging rather than gimmicky

## Tune

### Routes To Tune
- `hoboken -> grove-st`
  - now more spatially useful after the west-side downward shift, but still worth checking for over-connection
- `hoboken -> chelsea`
  - useful, but one of the more abstract west-to-Manhattan links
- `chelsea -> union-square`
  - currently okay, but could become a little too easy for central Manhattan cleanup
- `union-square -> battery-park`
  - already increased to `3`; keep watching whether Battery Park still rescues too much
- `newark-penn -> newark-airport`
  - still only length `2`; if Newark Airport starts reading as free value, this is the first southwest edge to price up
- `grand-central -> jamaica`
  - still a very efficient core-to-outer route that can overshadow Flushing if too cheap or too direct
- `midtown-west -> long-island-city`
  - still a strong center-to-Queens bridge and worth watching now that LIC -> Downtown Brooklyn is gone
- `long-island-city -> jamaica`
  - meaningful, but should be checked against the stronger Grand Central-based eastern trunks
- `battery-park -> downtown-brooklyn`
  - thematically good, but still part of the south-core ladder that can rescue many tickets

### Tickets To Tune
- `t-newark-grand-central`
  - still a central-trunk-heavy objective and should stay under scrutiny
- `t-world-trade-downtown-brooklyn`
  - may still reward the same south-core plan too often
- `t-battery-park-downtown-brooklyn`
  - now effectively a `Battery Park -> Downtown Brooklyn` ticket; useful, but can reinforce the same harbor ladder
- `t-battery-park-red-hook`
  - useful for validating Red Hook, but may need a point tweak depending on how often Red Hook matters
- `t-hudson-yards-grand-central`
  - correct local identity, but possibly a little too easy if both ends are already contested naturally
- `t-long-island-city-jamaica`
  - good Queens identity, but should be compared against the stronger Grand Central-based eastern tickets
- `t-newark-penn-district`
  - clean identity, but may be a little too easy now that Secaucus -> Newark Penn costs more and Newark still has multiple clean Manhattan paths
- `t-newark-airport-grove-st`
  - useful for supporting the airport, but should be checked to ensure it is not simply a cheap southwest local freebie
- `newark-airport -> hudson-yards`
  - flavorful and intentionally hard, but it should be watched to ensure it feels like a daring premium tunnel instead of an arbitrary shortcut

## Cut

### Routes Most Eligible To Cut If Another Simplification Pass Is Needed
- `hoboken -> chelsea`
  - first candidate if west side proves too forgiving or too abstract
- `midtown-west -> long-island-city`
  - not a first-choice cut, but now one of the more expendable central bridges if Manhattan/Queens overlap remains too high

### Tickets Most Eligible To Cut If Another Overlap Pass Is Needed
- `t-newark-grand-central`
  - strongest remaining candidate for central-trunk overlap pressure
- `t-world-trade-downtown-brooklyn`
  - if south-core overlap stays high, this is one of the first tickets to revisit
- `t-hudson-yards-grand-central`
  - lowest-cost west-Manhattan local ticket and easiest to cut if the deck needs sharper role clarity
- `t-newark-penn-district`
  - candidate if local central tickets feel too safe compared with outer-node tickets

## Current Reading
- `Battery Park` is still the main pivot to watch, but removing direct `Union Sq. -> World Trade` has made it less of a universal Manhattan shortcut.
- `Flushing` remains the healthiest recent addition and now has enough ticket support to justify staying.
- The biggest remaining overlap risk is no longer the old WTC trunk alone; it is the broader `Penn District / Grand Central / Battery Park` central economy.
- West side is cleaner and more realistic than before, but `Hoboken -> Chelsea` still looks like the first route to question if simplification is needed.
- The next changes should probably come from the `Tune` and `Cut` lists, not from adding more stations.
